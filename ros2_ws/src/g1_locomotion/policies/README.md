# G1 locomotion policy

This package bundles the Unitree pre-trained G1 locomotion TorchScript policy as `g1_motion.pt`.

Source:
- `deps/unitree_rl_gym/deploy/pre_train/g1/motion.pt`
- upstream repo: https://github.com/unitreerobotics/unitree_rl_gym

If you want to refresh the bundled copy, replace `g1_motion.pt` with the current upstream G1 locomotion export.
